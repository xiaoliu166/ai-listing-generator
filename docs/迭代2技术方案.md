# 第二个迭代技术方案

## 核心功能

### 1. 竞品 ASIN 分析
- **输入**: ASIN（支持批量最多5个）
- **处理**: 爬虫抓取 → 解析 → 关键词提取 → 可视化
- **输出**: 标题/五点/描述/评分 + 关键词高亮 + 词频图表

### 2. 评论分析
- **输入**: ASIN
- **处理**: 爬虫抓取评论 → 情感分析 → 卖点/痛点提取
- **输出**: 好评 Top 10 / 差评 Top 10 / AI 总结

### 3. 订阅制（简化版）
- 免费版：每天1次
- 付费版：无限次

---

## 技术选型

| 模块 | 技术 | 说明 |
|------|------|------|
| 爬虫 | Puppeteer | 抓取亚马逊页面 |
| AI 分析 | MiniMax API | 生成分析报告 |
| 存储 | 内存/文件 | 简化版先用本地存储 |
| 前端 | Next.js | 复用现有项目 |
