# AI Listing Generator - 可行性分析报告

> 生成时间：2026-03-10
> 参与角色：程序员、产品汪、小六（监督）
> 项目代号：AI Listing（跨境电商AI产品文案生成）

---

## 📋 一、项目概述

### 1.1 背景
跨境电商卖家上架一个产品需要：写标题、5点描述、关键词、详描文案——通常耗时 30-60 分钟/品。AI时代，这个工作可以大幅自动化。

### 1.2 目标
为跨境小白卖家提供低成本、高质量的 Amazon Listing AI 生成工具。

### 1.3 商业模式
- 免费版：10条/月
- 付费版：$9.9/月（无限生成）
- 目标用户：个人跨境卖家、小型团队

## 🏗️ 二、技术方案

### 2.1 技术栈

| 层级 | 技术选型 | 理由 |
|------|----------|------|
| 前端框架 | Next.js 14 | 全栈能力，部署简单 |
| UI框架 | Tailwind CSS | 轻量、简约清爽风格 |
| 状态管理 | Zustand | 简单够用 |
| 后端API | Next.js API Routes | 前后端一体 |
| LLM | MiniMax (abab6.5s-chat) | 国产API，便宜稳定 |
| 部署 | Vercel | 免费额度，CI/CD自带 |
| 代码托管 | GitHub | 团队熟悉 |

### 2.2 系统架构

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   用户端     │────▶│  Next.js   │────▶│  MiniMax    │
│  (浏览器)    │     │  API Route │     │  LLM API    │
└─────────────┘     └─────────────┘     └─────────────┘
                           │
                           ▼
                    ┌─────────────┐
                    │   Vercel    │
                    │   托管      │
                    └─────────────┘
```

### 2.3 核心功能

| 功能 | 优先级 | 说明 |
|------|--------|------|
| 单品生成 | P0 | 输入产品名+关键词 → 输出 Title + 5点 + Description |
| 风格选择 | P0 | 专业/休闲/高端/亲切 4种风格 |
| 一键复制 | P0 | 点击复制到剪贴板 |
| 次数限制 | P1 | 免费10条，付费无限 |
| 支付集成 | P2 | LemonSqueezy（后期） |

### 2.4 数据流

1. 用户输入产品信息（产品名、关键词、特点、风格）
2. 前端调用 `/api/generate` API
3. 后端构建 Prompt，调用 MiniMax API
4. MiniMax 返回 JSON 格式的 Listing
5. 前端渲染展示，支持一键复制

### 2.5 关键实现

#### Prompt 设计
```
角色：Amazon 产品文案专家
输入：产品名、关键词、产品特点、目标风格
输出：JSON格式（title, bullet1-5, description）
策略：融入关键词 + 强调用户利益 + 符合亚马逊规范
```

#### 成本控制
- MiniMax abab6.5s-chat: 约 ¥0.01/千字符
- 单次生成成本：约 ¥0.02-0.05
- 9.9美元/月仍有充足利润
